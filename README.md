## 一.信息素养的理解

### 1.*信息意识*

***使用信息技术解决工作和生活的问题的意识，这是信息技术教育中最重要的一点。***

### 2.信息尝试

***信息科学技术常识。***

### 3.信息能力

***包括***

- 信息系统的基本操作能力
- 信息的采集，传输，加工处理和应用能力
- 信息系统的分析与查错能力
- 对信息系统与信息进行评价的能力

### 4.信息道德

***培养学生具有正确的信息伦理道德修养，是他们能够遵循信息应用人员的伦理道德规范，不组织和参与非法活动，并知道如何防止计算机病毒和其他计算机犯罪活动。***
